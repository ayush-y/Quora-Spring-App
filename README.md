```Quora Spring Boot Clone

A backend implementation of a Quora-like platform built using Spring Boot.
This project demonstrates how a social discussion platform works internally including users, questions, answers, tags, and feeds.

The goal of this project is to practice backend architecture, REST APIs, database relationships, and scalable service design.

```Features

User Registration & Authentication

Create Questions

Post Answers

Tagging System

User Feed Generation

Follow Topics / Tags

CRUD operations for questions & answers

RESTful API architecture
